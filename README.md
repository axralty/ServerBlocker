# 🚫 ServerBlocker

Prevent accessing specific Discord servers entirely.

ServerBlocker lets you block servers so you **cannot open, join, or interact with them at all**. Navigation, invites, and even voice connections are automatically denied for blocked servers.

---

## ✨ Features

* 🚫 **Block access to selected servers completely**
* 🔒 **Prevents navigation into blocked servers**
* 📩 **Blocks joining via invite links**
* 🎧 **Prevents joining voice channels in blocked servers**
* 🔌 **Auto-disconnects if already connected**
* 🏷️ **On-screen banner when viewing a blocked server**
* 🖱️ **Floating button UI to manage blocklist**
* ➕ **Block current server with one click**
* 🧾 **Manual server ID input support**

---

## 📦 Installation

### 1. Install BetterDiscord

Download and install BetterDiscord:
👉 https://betterdiscord.app/

### 2. Download ServerBlocker

* Go to the **Releases** page
* Download the latest `ServerBlocker.plugin.js`

### 3. Move to Plugins Folder

Place the file in your BetterDiscord plugins folder:

```
Windows: %appdata%/BetterDiscord/plugins
Mac: ~/Library/Application Support/BetterDiscord/plugins
Linux: ~/.config/BetterDiscord/plugins
```

### 4. Enable Plugin

* Open Discord
* Go to **Settings → Plugins**
* Enable **ServerBlocker**

---

## 🧠 Usage

* Click the **floating button (bottom-right)** to open the panel
* Use **“Block Current Server”** to quickly block the server you're in
* Or manually add a server using its **Server ID**

Once blocked:

* You cannot open the server
* You cannot join via invite
* You cannot connect to voice channels

---

## ⚙️ Notes

* If you are already in a blocked server, you will be **redirected away automatically**
* If you are in a voice channel, you will be **forcefully disconnected**
* A **"BLOCKED SERVER"** banner will appear if detected
